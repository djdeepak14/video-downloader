## 📌 About The Project

This project is a **Video Downloader** built using **Python** and the **PyTube** library.  
It allows users to download YouTube videos directly to their local machine with just a link input.

You can use it as:
- A CLI tool
- A base for building a GUI downloader
- A learning project for Python networking and file downloads

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| 🐍 Python   | Main scripting language |
| 📦 PyTube   | YouTube video download library |
| 💾 OS & File System | Saving downloaded videos |

---

## 🚀 Features

✔ Simple command line interface  
✔ Download videos from YouTube by URL  
✔ Option to choose video resolution  
✔ Supports MP4 downloads  
✔ Lightweight and fast
